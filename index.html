¿Qué son las tecnologías Backend?

Backend es el termino para señalar la implementación de los lenguajes de programación del lado del servidor. Se encarga de la interacción web, almacenan la información en el servidor y se conectan a una base de datos para mostrar y actualizar su contenido conforme para el tiempo.

FronEnd son las que se desarrollan del lado del cliente o del lado del usuario, es decir, es todo con lo que el usuario mira y con lo que interactua
una buena experiencia de usuario, imersion y usabilidad son el objetivo de un desarrollador al diseñar el sitio web sumandole la existencia de frameworks y librerias virtuales que facilitan la vida del desarrollador frontend Para saber más sobre backend y frontend da clic aquí

Comparaciones ente Backedn y FrontEnd


Backend Frontend
se encargan de gestionar almacenamiento y la comunicación con la base de datos y el servidor Esta centrada unicamente en mejorar al experiencia del usuario
se usa JavaScript, HTML y CSS, además se debe dominar Python, Ruby, PHP, C++, etc. un desarrollador FronEnd solo necesita dominar frameworks Angular o React para desarrollar páginas enteras.
Está siempre oculto a la vista del usuario, el backend le dice que hacer al frontend. a menudo se sule navegar usando una GUI (interfaz gráfica de usuario) o una linea de comandos
Al ser del lado del servidor, a veces, solo renderiza en bases de datos que trabajan en segundo plano o un servidor de archivos que registra recursos para una empresa se intenta mejorar el rendimiento y capacidad de respuesta para que el sitio cargue rápido y funcione en cualquier dospositivo.
